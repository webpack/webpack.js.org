عملية نشر Webpack سهلة للغاية. اقرأ الخطوات التالية لتفهم كيفية القيام بذلك بوضوح.

## Pull Requests

عند دمج طلبات السحب (pull requests) في الـ`main`، حدد خيار _Create Merge Commit_.

## Releasing

```bash
npm version patch && git push --follow-tags && npm publish
npm version minor && git push --follow-tags && npm publish
npm version major && git push --follow-tags && npm publish
```

_سيؤدي هذا إلى زيادة رقم إصدار الحزمة، وتثبيت التغييرات، وإنشاء **local tag**، ورفع التغييرات إلى GitHub، ونشر حزمة npm._

بعد ذلك اذهب إلى صفحة الإصدارات على GitHub (https://github.com/webpack/webpack/releases) واكتب سجل التغييرات للـtag الجديدة.
