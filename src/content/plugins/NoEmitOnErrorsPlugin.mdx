يسمح لك `NoEmitOnErrorsPlugin` بمنع emit للـ assets عند وجود أخطاء. يكون مفعّلًا افتراضيًا، ويمكن تعطيله عبر [`optimization.emitOnErrors`](/configuration/optimization/#optimizationemitonerrors).

**webpack.config.js**

```js
export default {
  plugins: [new webpack.NoEmitOnErrorsPlugin()],
};
```
