`NoEmitOnErrorsPlugin`: تسمح لك بتفادي اضافة الأصول عندما يكون هناك اي أخطاء. مفعلة افتراضياً، يمكنك تعطيلها بأستخدام [`optimization.emitOnErrors`](/configuration/optimization/#optimizationemitonerrors)
**webpack.config.js**

```js
export default {
  plugins: [new webpack.NoEmitOnErrorsPlugin()],
};
```
