يفحص `AutomaticPrefetchPlugin` كل modules من compilation السابقة ويجلبها مسبقًا أثناء انتظار التغييرات، بهدف تحسين أوقات rebuild. يختلف ذلك عن [`PrefetchPlugin`](/plugins/prefetch-plugin/) الذي يجلب module واحدًا مسبقًا.

T> لا يُتوقع أن يقدم هذا plugin فائدة كبيرة للأداء في أغلب الحالات، لأن أوقات rebuild التزايدية تكون سريعة عادة.

**webpack.config.js**

```js
export default {
  // ...
  plugins: [new webpack.AutomaticPrefetchPlugin()],
};
```
