_السياق_ يشير الى [التعبيرات المنطقية الديناميكية في import() او require()](/guides/dependency-management/#dynamic-expressions-in-import-or-require) مثل `require('./locale/' + الاسم + '.json')`.
`ContextExclusionPlugin`: تسمح لك بإستبعاد السياق، ارفق تعبيرات نمطية  كـسبب عند تشغيل الإضافه لتستبعد كل السياق الذي يطابقها

**webpack.config.js**

```js
export default {
  plugins: [new webpack.ContextExclusionPlugin(/dont/)],
};
```
