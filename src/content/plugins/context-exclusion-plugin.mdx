يشير _context_ إلى [التعابير الديناميكية في `import()` أو `require()`](/guides/dependency-management/#dynamic-expressions-in-import-or-require)، مثل `require("./locale/" + name + ".json")`.

يسمح لك `ContextExclusionPlugin` باستبعاد ملفات من context عبر RegExp. مرر RegExp عند إنشاء plugin، وسيتم استبعاد كل request يطابقه.

**webpack.config.js**

```js
export default {
  plugins: [new webpack.ContextExclusionPlugin(/dont/)],
};
```
