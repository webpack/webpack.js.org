يفعّل [Hot Module Replacement](/concepts/hot-module-replacement)، المعروف اختصارًا بـ HMR.

W> لا يُنصح باستخدامه في production.

## الاستخدام الأساسي

تفعيل HMR مباشر غالبًا ولا يحتاج إلى خيارات إضافية.

```js
new webpack.HotModuleReplacementPlugin({
  // خيارات...
});
```
