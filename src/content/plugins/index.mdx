يمتلك webpack واجهة plugins قوية وغنية. كثير من ميزات webpack الداخلية مبنية على هذه الواجهة نفسها، وهذا ما يجعل webpack **مرنًا** وقابلًا للتوسع.

| الاسم                                                                           | الوصف                                                                               |
| ------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------- |
| [`BannerPlugin`](/plugins/banner-plugin)                                        | يضيف banner أعلى كل chunk يتم إنشاؤه                                                |
| [`ChunksWebpackPlugin`](/plugins/chunks-webpack-plugin/)                        | ينشئ ملفات HTML تربط entrypoints وchunks حتى تستطيع تقديم bundles                   |
| [`CompressionWebpackPlugin`](/plugins/compression-webpack-plugin)               | يجهز نسخًا مضغوطة من assets لتقديمها باستخدام Content-Encoding                      |
| [`ContextReplacementPlugin`](/plugins/context-replacement-plugin)               | يغيّر context الذي يستنتجه webpack من تعبير `require`                               |
| [`CopyWebpackPlugin`](/plugins/copy-webpack-plugin)                             | ينسخ ملفات مفردة أو مجلدات كاملة إلى مجلد البناء                                    |
| [`DefinePlugin`](/plugins/define-plugin)                                        | يعرّف constants عامة تُحقن وقت التجميع                                              |
| [`DllPlugin`](/plugins/dll-plugin)                                              | يقسم bundles لتحسين وقت البناء بشكل كبير                                            |
| [`EnvironmentPlugin`](/plugins/environment-plugin)                              | اختصار لاستخدام [`DefinePlugin`](/plugins/define-plugin) مع مفاتيح `process.env`    |
| [`EslintWebpackPlugin`](/plugins/eslint-webpack-plugin)                         | plugin لتشغيل ESLint مع webpack                                                     |
| [`HotModuleReplacementPlugin`](/plugins/hot-module-replacement-plugin)          | يفعّل Hot Module Replacement (HMR)                                                  |
| [`HtmlWebpackPlugin`](/plugins/html-webpack-plugin)                             | ينشئ ملفات HTML بسهولة لتقديم bundles                                               |
| [`IgnorePlugin`](/plugins/ignore-plugin)                                        | يستبعد modules محددة من bundles                                                     |
| [`LimitChunkCountPlugin`](/plugins/limit-chunk-count-plugin)                    | يضع حدًا أدنى أو أعلى لعدد chunks لتحكم أفضل في التقسيم                             |
| [`MergeDuplicateChunksPlugin`](/plugins/merge-duplicate-chunks-plugin)          | يدمج chunks التي تحتوي modules نفسها                                                |
| [`MinChunkSizePlugin`](/plugins/min-chunk-size-plugin)                          | يحافظ على حجم chunk فوق حد معين                                                     |
| [`MiniCssExtractPlugin`](/plugins/mini-css-extract-plugin)                      | ينشئ ملف CSS لكل ملف JS يحتاج إلى CSS                                               |
| [`NoEmitOnErrorsPlugin`](/configuration/optimization/#optimizationemitonerrors) | يتجاوز مرحلة الإخراج عندما توجد أخطاء تجميع                                         |
| [`NormalModuleReplacementPlugin`](/plugins/normal-module-replacement-plugin)    | يستبدل الموارد التي تطابق regexp                                                    |
| [`ProgressPlugin`](/plugins/progress-plugin)                                    | يعرض تقدم عملية التجميع                                                             |
| [`ProvidePlugin`](/plugins/provide-plugin)                                      | يستخدم modules بدون الحاجة إلى `import` أو `require` في كل ملف                      |
| [`SourceMapDevToolPlugin`](/plugins/source-map-dev-tool-plugin)                 | يوفر تحكمًا أدق في source maps                                                      |
| [`EvalSourceMapDevToolPlugin`](/plugins/eval-source-map-dev-tool-plugin)        | يوفر تحكمًا أدق في eval source maps                                                 |
| [`SvgChunkWebpackPlugin`](/plugins/svg-chunk-webpack-plugin/)                   | ينشئ SVG sprites محسّنة بواسطة SVGO اعتمادًا على dependencies الخاصة بـ entry point |
| [`MinimizerPlugin`](/plugins/minimizer-webpack-plugin/)                         | يستخدم Terser أو غيره لتصغير JS/CSS/HTML/JSON وغيرها داخل مشروعك                    |

لمزيد من plugins الخارجية، راجع قائمة [awesome-webpack](/awesome-webpack/#webpack-plugins).
