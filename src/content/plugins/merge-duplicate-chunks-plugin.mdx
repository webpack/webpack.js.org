دمج الاجزاء التي تحتوي على نفس المودلات، هذه الإضافه مفعلة افتراضياً وهي جزء من تحسينات webpack الداخلية. استخراجها علناً يسمح للمطورين بالتحكم بالـ`مرحلة (stage)` عند ماذا يحصل الدمج — مفيدة عندما تدمج الاجزاء المطلوبة للتشغيل بعد [`إضافة تقسيم الاجزاء`](/plugins/split-chunks-plugin)

```js
new webpack.optimize.MergeDuplicateChunksPlugin({
  // Options...
});
```

## الخيارات

### stage

`رقم`

يتحكم بتحسين المرحلة، عند أي مرحلة يتم تشغيل هذه الإضافة، إفتراضياً, يحصل الدمج قبل `إضافه تقسيم الاجزاء`. وضع رقم اعلى للـ`مرحلة (stage)` يجعل الدمج يعمل بعد تحسينات اخرى مثل  `إضافه تقسيم الاجزاء`

**webpack.config.js**

```js
import webpack from "webpack";

export default {
  // ...
  plugins: [
    new webpack.optimize.MergeDuplicateChunksPlugin({
      stage: 10, // يعمل بعد اضافة تقسيم الاجزاء
    }),
  ],
};
```
