يدمج chunks التي تحتوي على modules نفسها. هذا plugin مفعّل افتراضيًا وجزء من تحسينات webpack الداخلية. وجوده كـ plugin مستقل يسمح للمطورين بالتحكم في `stage` التي يحدث عندها الدمج، وهذا مفيد إذا أردت تشغيل الدمج بعد [`SplitChunksPlugin`](/plugins/split-chunks-plugin).

```js
new webpack.optimize.MergeDuplicateChunksPlugin({
  // Options...
});
```

## الخيارات

### stage

`number`

يتحكم في مرحلة تشغيل هذا التحسين. افتراضيًا يحدث الدمج قبل `SplitChunksPlugin`. استخدام رقم أعلى للـ `stage` يجعل الدمج يعمل بعد تحسينات أخرى، مثل `SplitChunksPlugin`.

**webpack.config.js**

```js
import webpack from "webpack";

export default {
  // ...
  plugins: [
    new webpack.optimize.MergeDuplicateChunksPlugin({
      stage: 10, // يعمل بعد SplitChunksPlugin
    }),
  ],
};
```
