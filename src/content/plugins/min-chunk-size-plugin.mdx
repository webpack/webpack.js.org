ابقاء حجم الاجزاء فوق الحد المعين عن طريق دمج الاجزاء التي تكون اصغر من `minChunkSize`.

```js
new webpack.optimize.MinChunkSizePlugin({
  minChunkSize: 10000, // اقل عدد من الحروف
});
```

## الأستعمال عن طريق واجهة الاوامر (CLI)

هذه الاضافه واعداداتها يمكن التحكم فيها وادارتها عن طريق واجهة الأوامر  [CLI](/api/cli/):

```bash
webpack --optimize-min-chunk-size 10000
```
