يحافظ على حجم chunks فوق الحد المحدد عبر دمج chunks الأصغر من `minChunkSize`.

```js
new webpack.optimize.MinChunkSizePlugin({
  minChunkSize: 10000, // أقل حجم مسموح به للـ chunk
});
```

## الاستخدام عبر CLI

يمكن التحكم بهذا plugin وخياراته عبر [CLI](/api/cli/):

```bash
webpack --optimize-min-chunk-size 10000
```
