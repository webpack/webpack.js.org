جلب مسبق اعتيادي لطلبات المودل، مما يؤدي الى حلها وبنائها قبل حدوث اول عملية `import` او `require` لتلك المودل، استخدام هذه الإضافه يمكن ان يسرع الاداء. حاول تحليل البنية اولاً لتحديد نقاط الجلب المسبق الذكية

```js
import webpack from "webpack";

new webpack.PrefetchPlugin([context], request);
```

## الخيارات

- `context`: مسار حتمي لمجلد
- `request`: طلب من مودل عادية
