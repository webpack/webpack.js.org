يجلب request الخاص بـ module مسبقًا، بحيث يتم resolve وبناء module قبل أول `import` أو `require` له. قد يسرّع هذا الأداء، لكن يفضل تحليل بنية التطبيق أولًا لتحديد نقاط prefetch المفيدة.

```js
import webpack from "webpack";

new webpack.PrefetchPlugin([context], request);
```

## الخيارات

- `context`: مسار مطلق إلى مجلد.
- `request`: request عادي لـ module.
