يتجاهل ملفات محددة أثناء [watch mode](/configuration/watch)، مثل الملفات التي تطابق مسارًا أو RegExp معينًا.

```js
new webpack.WatchIgnorePlugin({ paths });
```

## الخيارات

- `paths` (`Array<string | RegExp>`): قائمة RegExp أو مسارات مطلقة إلى مجلدات أو ملفات يجب تجاهلها.

T> تم إهمال هذا plugin واستبداله بخيار [`watchOptions.ignored`](/configuration/watch/#watchoptionsignored).
